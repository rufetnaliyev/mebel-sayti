## Common Customizations

### Change Colors
Edit `css/style.css`, find `:root` section:
```css
--primary-color: #2C5F2D;    /* Main green */
--accent-color: #C9A961;     /* Gold accent */
```

### Update WhatsApp Number
Search and replace `995552309189` in all files

### Add Product Images
Replace:
```html
<div class="placeholder-image">...</div>
```
With:
```html
<img src="../images/product.jpg" alt="Product Name">
```
